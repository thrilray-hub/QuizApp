# QuizApp
QuizApp is a responsive and interactive digital tool used to test knowledge, training, or engage audiences through a series of questions and answers. Built using HTML, CSS, and JavaScript. The application allows users to answer multiple-choice questions and receive a score at the end of the quiz.
This project demonstrates DOM manipulation, event handling, and dynamic UI updates, which are key skills for frontend development.
Live Demo
https://thrilray-hub.github.io/QuizApp/

Features
•	Multiple-choice quiz questions
•	Instant answer selection
•	Score calculation at the end of the quiz
•	Dynamic question loading
•	Responsive design for mobile and desktop
•	Restart quiz functionality

Technologies Used
•	HTML5 – Structure of the application
•	CSS3 – Styling and layout
•	JavaScript (ES6) – Quiz logic and DOM manipulation

How It Works
1.	The quiz loads a question and multiple answers.
2.	The user selects an answer.
3.	JavaScript checks whether the answer is correct.
4.	The next question is displayed.
5.	After the last question, the final score is shown.
